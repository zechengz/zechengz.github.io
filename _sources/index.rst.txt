DeepSNAP Documentation
====================================

DeepSNAP is a Python package that makes the implementation of deep learning on graphs with transformations easy.

It connects several Python packages such as `NetworkX <https://networkx.github.io/>`_, `PyTorch Geometric <https://pytorch-geometric.readthedocs.io/en/latest/>`_, and `PyTorch <https://pytorch.org/>`_.
